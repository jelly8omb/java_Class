복습 QnA

1. 메서드 정의 - 왜 만드는 것? (다른 프로그래밍 언어에서 `함수`와 같은 동작)
    - 자주 사용되는 기능을 main 에서 분리시킨다.
    - 코드를 여러번 작성하지않고 재사용 가능.

2. 메서드의 인자, 매개변수란 무엇인가?
    - 인자(argument) : 둘 사이의 값을 연결시킨다. 
                        메서드 호출시 입력값을 메서드 정의한 실행 부분으로 연결시켜준다.
    - 매개변수 : 선언된 변수. () 안에서 선언된 변수 그 자체를 말합니다.
    - 결론 - : 인자와 매개변수는 같은 의미이다.

3. 메서드의 리턴은 무엇인가?
    - 메서드 실행한 결과(출력)값.

4. static 메서드의 특징은 무엇인가?
        답 : 객체를 생성하지 않고 클래스 이름으로 직접 실행(호출)
        예시 : Character.isUpperCase(c) String.format("%d", val) Integer.valueOf("123")
    (비교) 인스턴스 메서드는 객체를 만들어서 메서드를 호출
    (인스턴스 메서드 예시) message.length(), sc.nextInt()


** 정리 ** : 메서드는 인스턴스 메서드와 static 메서드로 분류할 수 있습니다.
