1. 인스턴스 필드에 대한 설명으로 옳은가

인스턴스 필드는 객체의 행동을 정의하며, 특정 인스턴스에 대해 작동합니다. 
인스턴스 필드는 일반적으로 객체의 상태를 조작하거나 객체에 대한 연산을 수행하는 데 사용됩니다.
1. O                 2. X

2. Setter에 대한 설명으로 옳은가

Setter(세터)는 객체의 속성 값을 설정하거나 변경하는 메소드를 말합니다. 
일반적으로 객체의 속성은 클래스 내부에서 정의되어 있고, 이 속성에 값을 설정하기 위해 Setter 메소드가 사용됩니다. 
Setter 메소드는 객체의 내부 속성을 변경하기 위해 사용되며, 
속성을 외부에서 직접적으로 수정하지 않고 메소드를 통해 값을 설정합니다.
1. O                    2. X
